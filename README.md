# EMS Gems Jewelry Store

## Overview
This project is a jewelry store web application built with ASP.NET Core Razor Pages. It was created as a final project to simulate an online storefront experience and demonstrate web development, database integration, and application structure.

## Features
- Product display and listings
- Store-style layout and navigation
- Database integration using SQLite
- Session support
- Structured Razor Pages application

## Technologies Used
- C#
- ASP.NET Core Razor Pages
- Entity Framework Core
- SQLite
- HTML
- CSS
- JavaScript

## My Role
I designed and developed this project, focusing on layout, usability, database connection, and building a structured web application.

## How to Run
1. Open the project in Visual Studio
2. Restore dependencies
3. Run the application
4. The app uses SQLite through Entity Framework Core and loads product data through the configured database connection
